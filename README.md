1.CLASSFIER是我们以一定逻辑搭建的数据集标注程序，以退为进，在应对极其难以处理的样本时，规避极其复杂的临床诊断逻辑选择直接将疾病对应到具体样本，牺牲部分样本数量简化程序逻辑。
  相关判断逻辑：
    以如下图片标注逻辑编写python代码完成图片标注任务并且检验程序是否正常运行：
    1.读取xlsx文件，文件中有多列，其中第四、五列是图片文件名，第六、七列是图片的性质（normal fundus或其他），第八到十五列是独热编码（8个类别）。
    2.我们要创建8个独热编码对应的文件夹（即8个类别），以及一个额外的"N"文件夹（用于存放normal fundus的图片）。
    3.对于每一行，先检查独热编码部分（第八到十五列）是否只有一个1。如果不是（即多个1或没有1），则忽略这一行，不处理对应的两张图片。
    4.如果独热编码部分只有一个1，那么：
    a. 如果第六列和第七列的值都是"normal fundus"，那么这两张图片都放到"N"文件夹。
    b. 否则，对于这两张图片，检查它们对应的性质（第六列对应第四列的图片，第七列对应第五列的图片）：
    如果性质是"normal fundus"，则将该图片放入"N"文件夹；
    如果性质不是"normal fundus"，则将该图片放入独热编码所指示的类别文件夹中。
    P.S:class1和N都表示该眼底照片正常
